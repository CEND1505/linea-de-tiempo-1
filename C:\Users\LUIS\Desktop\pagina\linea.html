LINEA DEL TIEMPO
PROGRAMACION WEB

1960

Esta web estaba diseñada solo para lectura, el usuario no podía interactuar con el contenido de las paginas. La información era totalmente limitada a lo que el webmaster publicaba, es decir, unidireccional NO colaborativa.

1961

Se publican en internet los primeros ficheros del primer navegador web

1970

Se completa Network Control Protocol (NCP), el primer protocolo host-to-host empleado en ARPANET

1978

TCP se separa de TCP/IP: TCP se encarga de la comunicacion extremo a extremo e IP sel proceso de direccionamiento

1990

Se instala el primer servidor web y se publica la primera pagina web

1991

Surgió el HTML el cual hizo las páginas web más agradables a la vista. Tim Berners-Lee fue quien creó HTML,el lenguaje empleado para crear los documentos de la Web, basado en Standard Generalized Markup Language (SGML),HyperText Transfer Protocol (HTTP), el protocolo que emplean los ordenadores para comunicarse en la Web, y Universal Resource Locator (URL), como medio de localización de los distintos recursos que forman la Web en Internet.

1993

Navegador creado en la Universidad de Illinois por Marc Andreessen y que superaba a todos al permitir, por ejemplo, incluir imágenes en las páginas web.

1994

La compañía de software Netscape lanzó primera versión de su navegador comercial para internet, esta estuvo disponible de forma gratuita para todos los usuarios ya que, dentro de las políticas de software, estaba la idea de que el buscador para internet no debería tener ningún costo

1995

WEB 2.0 También conocida como "WEB SOCIAL O COLABORATIVA". - Este nuevo formato fomentaba la participación activa del usuario, ofrecía dinamismo y su principal característica era que el usuario no solo accedía a información sino que también podía crear contenido.

2001

W3C (World Wide Web) creó grupos de trabajo dedicados al desarrollo de la Web Semántica, un conjunto de tecnologías para describir y relacionar la información contenida en una página web para su análisis y procesamiento automático.

2003

WEB 3.0 Jeffrey Zeldman fue quien nombro por primera vez la web 3.0. Esta web se basa en la información y su estructuración, permitiéndole al usuario encontrar datos de manera rápida, eficaz, contenidos interactivos, donde se puedan asociar temas, noticias, investigación, entre otras, y la información obtenida sea de uso público. Se caracteriza por contenido accesible a través de múltiples dispositivos, en cualquier lugar y momento.

2020

Web 4.0 Creación de asistentes virtuales o agentes web que tengan la capacidad de conocer, aprender y razonar como las personas. Dispositivos delgados, mas ligeros y portátiles con altas resoluciones. Ubicación GPS
